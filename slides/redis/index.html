OK

"pong"

OK

OK

Time to Live


127.0.0.1:6379> SETEX 5 ethan "votre mission si vous l'acceptez..."
OK
      
127.0.0.1:6379> TTL ethan
5
      
127.0.0.1:6379> GET ethan
"votre mission si vous l'acceptez..."
      
# 5' later
127.0.0.1:6379> GET ethan
(nil)
